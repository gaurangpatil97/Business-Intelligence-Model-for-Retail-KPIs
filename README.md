## üìä Project Overview

Retail organizations often struggle with scattered data and inconsistent performance tracking.  
This project builds an end-to-end **Business Intelligence pipeline** that unifies sales, marketing, and CRM data into actionable insights through predictive modeling and KPI visualization.

---

## üß© Key Features

- **KPI Analytics:** Aggregates and models key business performance indicators across multiple dimensions (region, category, seasonality).  
- **Customer Segmentation:** Implements **RFM Analysis** and **K-Means Clustering** to identify high-value customer groups.  
- **Churn Prediction:** Trains a **Random Forest Classifier** achieving ~85% accuracy for churn risk detection.  
- **Cross-Selling Engine:** Uses **Apriori association rules** for product recommendation and bundle optimization.  
- **Sales Forecasting:** Predicts category-level demand using **Random Forest Regressor** and time-series analysis.  
- **Interactive Visualization:** Generates dashboards and correlation heatmaps for performance review.

## üß† Insights & Outcomes
